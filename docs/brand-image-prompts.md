# WorkCue Brand Image Prompts

Use these prompts to generate a hero image, repository icon, or social preview for WorkCue. Avoid putting real company names, personal data, real tickets, or private URLs in generated images.

## Hero Image

```text
Create a polished hero image for an open-source developer tool named WorkCue. The product is a local-first AI morning planner that reads existing work tools like GitHub, Jira, Obsidian, and Markdown notes, then recommends today's top work with evidence. Show a calm dark-mode command center at the start of a workday: a terminal window with a concise morning brief, small source cards for GitHub PRs, Jira issues, and Obsidian notes flowing into a single prioritized focus list. The mood should be focused, trustworthy, privacy-conscious, and developer-native. Use a modern SaaS/open-source aesthetic, crisp typography, subtle teal/blue/green accents, and clean depth. No real brand logos, no real ticket IDs, no private data, no people, no clutter. 16:9 aspect ratio, high resolution, suitable for a GitHub README hero.
```

## Repository Icon

```text
Design a simple app icon for WorkCue, a local-first AI morning planner for developers and teams. Visual metaphor: a sunrise cue card, a checkmark, and connected work signals converging into one focused path. Use a rounded square icon, dark navy background, teal and blue highlights, minimal vector-like shapes, high contrast, recognizable at small sizes. No text, no real company logos, no mascots, no clutter.
```

## Social Preview

```text
Create a GitHub social preview image for WorkCue. Include the title "WorkCue" and the tagline "Wake up to the right work." Show a clean dark-mode interface with a morning brief, evidence bullets, and small abstract source chips labeled GitHub, Jira, Obsidian, MCP, Markdown. The composition should make it clear that existing tools remain the source of truth and WorkCue is the morning intelligence layer. Use professional open-source launch aesthetics, crisp layout, restrained colors, and strong readability. No real logos, no private data, no exaggerated AI imagery. 1200x630.
```

```text

Show fragmented work signals from GitHub pull requests, Jira issues, Obsidian notes, and Markdown tasks being gently organized into three ranked cards: Focus, Watchlist, Not Today. The style should be clean, technical, flat-to-semi-3D, dark mode, restrained gradients, and suitable for an open-source productivity CLI/MCP project. No real logos, no private data, no screenshots of actual products, no people. Transparent or dark background, high resolution.
